Companion object AsyncAwaitDsl

class AsyncAwaitDsl[F[_]] extends AnyRef

WARNING: This construct currently only works on scala 2 (2.12.12+ / 2.13.3+), relies on an experimental compiler feature enabled by the -Xasync scalac option, and should absolutely be considered unstable with regards to backward compatibility guarantees (be that source or binary).

Partially applied construct allowing for async/await semantics, popularised in other programming languages.

object dsl extends AsyncAwaitDsl[IO]
import dsl._

val io: IO[Int] = ???
async { await(io) + await(io) }

The code is transformed at compile time into a state machine that sequentially calls upon a Dispatcher every time it reaches an "await" block.

  7.  def await[T](awaitable: F[T]): T

    Non-blocking await the on result of awaitable.

    Non-blocking await the on result of awaitable. This may only be used directly within an enclosing async block.

    Internally, this transforms the remainder of the code in enclosing async block into a callback that triggers upon a successful computation outcome. It does *not* block a thread.
